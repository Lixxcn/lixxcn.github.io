# 智算网络运维方向赛事大纲

## 命题大纲

**二〇二五年六月**

### 一、参赛对象说明

此赛项仅设置职工组，主要面向从事智算网络运维领域的信息通信网络运行管理员等相关专业或职业的企业职工。

### 二、赛事内容大纲

赛事内容大纲的设计原则：

（一）围绕参赛人员的岗位能力模型，覆盖能力要求，牵引人才培养，“以赛促学、以赛促训”；

（二）聚焦于“智算网络运维”领域，注重理论和实操相结合，保持内容的实用性、先进性。

赛事内容主要包括智算网络、人工智能与模型、智能运维三方面。
